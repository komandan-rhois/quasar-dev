# Quasar CRUD TypeScript


## Install =dependencies
```bash
yarn
```

### Start development mode
```bash
quasar dev
```

### Lint  file
```bash
yarn run lint
```

### Build  production
```bash
quasar build
```

### Customize
See [Configuring quasar.conf.js](https://quasar.dev/quasar-cli/quasar-conf-js).
